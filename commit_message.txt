⚡ Bolt: Performance and accessibility optimizations

💡 What: Replaced setInterval with recursive setTimeout and requestAnimationFrame, added aria-label to buttons, and optimized dictionary lookups.
🎯 Why: To prevent API piling, fix accessibility issues for screen readers on buttons without text, and improve performance in tight loops fetching coingecko prices.
📊 Impact: Prevents backend DoS, improves screen reader support, and reduces overhead on portfolio value fetching.
🔬 Measurement: Verified with tests and UI inspection.
